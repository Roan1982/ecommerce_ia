⚠️ Función Avanzada - Solo para Administradores

Los ContentTypes normalmente se crean automáticamente por Django cuando defines modelos en tu código. Crear ContentTypes manualmente puede causar problemas en el sistema si no se hace correctamente.

Usa esta función solo si:

  • Sabes exactamente lo que estás haciendo
  • El ContentType fue eliminado accidentalmente
  • Estás migrando datos de otro sistema
Información Técnica
Campos requeridos:
  • app_label: Nombre de la aplicación (ej: 'tienda', 'auth')
  • model: Nombre del modelo en minúsculas (ej: 'producto', 'user')
Campos generados automáticamente:
  • name: Nombre legible del modelo (generado automáticamente)
  • id: Identificador único (generado automáticamente)
Consideraciones:
  • El modelo debe existir en el código Python
  • Los permisos se crean automáticamente
  • No modifiques ContentTypes existentes
